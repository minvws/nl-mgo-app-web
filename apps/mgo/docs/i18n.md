# i18n (Internationalization)

For the translation of content we use the [Format.JS / React-intl library][1]. We don't manage the translations inside this repository. Instead, all the content is managed using a [lokalise][2] account.

## Updating the messages

For any changes to be made to the copy, you will need to do so in the [lokalise][2] UI. After you have made the neccessary changes there you can download the latest translations [using the lokalise web interface][4].

Use these settings:

-   File format: JSON
-   Languages: all
-   File structure: Multiple files per language (use assigned filenames). Directory prefix: `%LANG_ISO%`
-   Plural format: ICU
-   Placeholder format: ICU

place the download content in the `i18n/raw` folder and run these commands:

```
pnpm run messages:compile
pnpm run messages:format
```
