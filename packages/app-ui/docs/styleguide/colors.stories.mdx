# Styleguide - Colors

All the available color (palettes). To apply a color to an element you can use the tailwind utility classes. For example:

<Source
    language="html"
    code={`
<div class="bg-white text-grey-900" />
`}
/>

<ColorPalette style={{ wordBreak: 'break-all' }}>
    {singleColors.map(([name, colors]) => (
        <ColorItem key={name} title={`${name}`} subtitle={`${name} color scheme`} colors={colors} />
    ))}
    {colorSwatches.map(([name, colors]) => (
        <ColorItem key={name} title={`${name}`} subtitle={`${name} color scheme`} colors={colors} />
    ))}
</ColorPalette>
