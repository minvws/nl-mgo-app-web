<h1 align="center">Mijn Gezondheidsoverzicht</h1>
<br />

This package contains the MGO main application. It is a [React](https://react.dev/) based web application built using [vite](https://vitejs.dev/).

## Running a local development server

First ensure you have installed the dependencies. Then you can start up the development server.

```sh
pnpm install
pnpm dev
```
