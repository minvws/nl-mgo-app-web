# Welcome to the App UI Library

Here you will find a collection of components and styles to use for your project.

<br />

## Components

All components of the `app-ui` are available through the index. Which means you can directly import and start using them without specifying any extra paths (_provided that the alias paths are configured correctly_).

<Source
    language="ts"
    code={`
import { Heading } from '@minvws/pgo-app-ui';
`}
/>

All the available components are documented here with examples and controls which you can use to test them.

<br />

## Tailwind

The ui-library also comes with a [tailwind](https://tailwindcss.com/) config which is set up to match [the design](<https://www.figma.com/file/IGdxtARM9ssEvgRRMy5xxt/Mijn-Gezondheids-Overzicht-(realisatie)?type=design&node-id=344-3405&mode=design&t=GwMD9QoTBG2HNtWd-0>). It contains the [fundamentals](https://www.figma.com/file/aFJlQTQ4XzGF18OsiRVqJa/%F0%9F%8E%A8-Fundamentals) to be used for all the styles.

Tailwind offers a great [VSCode plugin](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) which can read the config and help with highlighting / linting and autocompletion. We highly recommend you install this plugin.
There are a lot of utility helpers included which [are well documented](https://tailwindcss.com/docs/utility-first).
You use the styles by simply applying the correcty utility class names to an element.
