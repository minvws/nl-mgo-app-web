# [MGO] - i18n (Internationalization)

For managing the all the copy and their translations, we make use of a [lokalise][lokalise] account. Here we maintain several files containing the copy of different parts of the MGO app and the FHIR data.

For the implementation of the copy we use the [Format.JS / React-intl library][format-js].

## Updating the messages (apps/mgo)

For any changes to be made to the copy, you will need to do so in the [lokalise][lokalise] UI. After you have made the neccessary changes there you can download the latest translations [using the lokalise web interface][lokalise-dl].

Use these settings:

- File format: JSON
- Languages: all
- File structure: Multiple files per language (use assigned filenames). Directory prefix: `%LANG_ISO%`
- Plural format: ICU
- Placeholder format: ICU

place the download content in the `i18n/raw` folder and run these commands:

```
pnpm run messages:compile
pnpm run messages:format
```
